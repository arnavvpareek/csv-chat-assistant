# LLM-Powered Data Analysis Assistant

A Streamlit chatbot that takes any uploaded CSV and answers natural-language
questions about it by generating and running pandas code — showing the code
alongside every answer.

**Stack:** Python · LangChain · Pandas · Groq · Streamlit
